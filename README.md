## 📌 시작하기

1. 저장소 클론  
   ```bash
   git clone https://github.com/sally03915/git0.git
   ```

2. 프로젝트 폴더 열기  
   ```bash
   cd git0  
   code .
   ```

3. 파일 수정 및 커밋  
   ```bash
   git add .  
   git commit -m "파일 수정 완료"  
   git push origin main
   ```

> 실시간 커밋과 브랜치 전략을 통해  
> 협업 환경에서의 코드 품질과 커뮤니케이션 효율을 높입니다.
